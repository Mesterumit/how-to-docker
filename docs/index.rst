Docker for AI/ML Bootcamp
==========================

Welcome! This is a hands-on introduction to containerization designed specifically for machine learning and data science bootcamp students.

**What you'll build**: Through three progressive examples (30-45 minutes), you'll containerize data processing scripts, web applications, and finally create your own publishable ML development environment.

.. note::
   
   **Ready to start?** Head to :doc:`introduction/overview` to begin your journey into containerization.
